🚀 Cách chạy ứng dụng

  1. Tải về 3 file: index.html, style.css, script.js và đặt cùng một thư mục
  2. Mở terminal tại thư mục đó, chạy: npx serve . hoặc dùng VS Code Live Server
  3. Truy cập http://localhost:3000 trên trình duyệt
  4. Nhấn "Bật Camera" → hệ thống sẽ tải mô hình Teachable Machine

🗄 Lưu trữ vào Google Sheets (khuyến nghị)

Bước 1: Tạo Google Sheet mới → Extensions → Apps Script

function doPost(e) {
  const sheet = SpreadsheetApp.getActiveSheet();
  const data = JSON.parse(e.postData.contents);
  sheet.appendRow([
    data.timestamp,
    data.type,
    data.confidence,
    data.userGuess,
    data.isCorrect
  ]);
  return ContentService.createTextOutput("OK");
}

Bước 2: Deploy → New deployment → Web App → Anyone

Bước 3: Copy URL và dán vào biến SHEETS_URL trong script.js

🔥 Lưu trữ vào Firebase Realtime Database

// Trong script.js, thêm:
import { initializeApp } from "firebase/app";
import { getDatabase, ref, push } from "firebase/database";

const firebaseConfig = { /* config của bạn */ };
const app = initializeApp(firebaseConfig);
const db = getDatabase(app);

function saveToFirebase(entry) {
  push(ref(db, 'waste-logs'), entry);
}
